# Hands on Health Chiropractic — Website Redesign

Modern, fast, responsive redesign of [hohchiro.com](https://hohchiro.com) built with Next.js 15, TypeScript, and Tailwind CSS.

## Tech Stack

- **Next.js 15** (App Router, static export)
- **TypeScript**
- **Tailwind CSS v4**
- **Lucide React** icons
- **Montserrat** font (Google Fonts)

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build for Production

```bash
npm run build
```

Static export outputs to `/out` — ready to deploy to Vercel, Netlify, or any static host.

## Pages

- `/` — Homepage (hero, services, testimonials, patient center, map)
- `/about` — About the practice & team
- `/services` — General services (adjustments, decompression, massage)
- `/other-services` — Other services (nutrition, laser, supplements)
- `/videos` — Video gallery (placeholder embeds)
- `/contact` — Contact info, hours, and map

## What's Better Than the Current Site

1. **Modern horizontal nav** — no more wasted space from vertical sidebar
2. **Mobile-first responsive design** — works great on all devices
3. **Fast static site** — no WordPress overhead, loads in under 1 second
4. **Clean, professional layout** — same brand colors, way better UX
5. **Proper CTAs** — clear scheduling buttons throughout every page
6. **SEO-optimized** — semantic HTML, proper heading hierarchy, meta tags
7. **No annoying popups** — clean first impression
8. **Easy to maintain** — simple component-based architecture
